# Gemini Review Prompt Template

Copy this prompt when asking Gemini to review a design.

---

## Standard Review Prompt

```
I'd like your visual design critique on this [pitch document / website design].

**Context:**
- Target audience: [e.g., local service businesses]
- Purpose: [e.g., sales one-pager to book discovery calls]
- Brand feel: [e.g., premium, trustworthy, modern]

**What Claude Code built:**
[Attach screenshot or describe key elements]

**Please provide:**

1. **Visual Polish** (3 specific things)
   - What micro-details would you refine?
   - Spacing, alignment, color balance, typography?

2. **What Feels Off?**
   - Anything that catches your eye negatively?
   - Does the hierarchy work?

3. **Alternative Ideas** (1-2 suggestions)
   - What would you try differently?
   - Any fresh directions worth exploring?

Keep feedback actionable. I'll implement the best suggestions.
```

---

## Quick Review (Faster)

```
Quick visual critique of this design:

[Attach screenshot]

Context: [one sentence about purpose]

Give me:
- 3 polish suggestions
- 1 alternative idea

Be specific and actionable.
```

---

## Deep Review (More Thorough)

```
Deep design review needed.

**The Design:**
[Attach screenshot + brief description]

**Context:**
- Audience: [who]
- Goal: [what action we want]
- Constraints: [any limitations]

**Review Areas:**

1. **First Impression (3 seconds)**
   - What do you notice first?
   - Is the hierarchy clear?

2. **Visual Design**
   - Color harmony
   - Typography choices
   - Spacing and rhythm
   - Visual balance

3. **Emotional Impact**
   - What feeling does it evoke?
   - Does it match the intended brand?

4. **Conversion Elements**
   - Is the CTA clear?
   - Trust signals present?
   - Friction points?

5. **Alternative Directions**
   - 2-3 different approaches worth exploring
   - Brief sketch of each idea

Be direct. I want honest critique, not validation.
```

---

## Notes

- Always include a screenshot (Gemini is visual)
- Give enough context but don't over-explain
- Ask for specific number of items (3, not "some")
- Mention you'll implement suggestions (focuses the feedback)
